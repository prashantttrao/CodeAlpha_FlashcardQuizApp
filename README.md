# CodeAlpha_FlashcardQuizApp

Flashcard Quiz App for the CodeAlpha internship task.

## What the App Does

- Shows one flashcard at a time.
- Displays the question first and reveals the answer with a "Show Answer" button.
- Lets users move with "Previous" and "Next".
- Lets users add, edit, and delete flashcards.
- Saves flashcards on the phone using local storage.
- Uses a modern, minimal native Android UI with a polished flashcard layout.

## How to Run in Android Studio

1. Open Android Studio.
2. Click **Open**.
3. Select this folder:

   `CodeAlpha_FlashcardQuizApp`

4. Wait for Gradle sync to finish.
5. Click the green **Run** button.
6. Choose an emulator or connected Android phone.

## How to Make the APK

1. Open the project in Android Studio.
2. Click **Build** in the top menu.
3. Click **Build App Bundle(s) / APK(s)**.
4. Click **Build APK(s)**.
5. When Android Studio finishes, click **Locate** to open the APK file.

If Android Studio says the Android SDK is missing, open **Tools > SDK Manager**, install the recommended SDK, then build again.
